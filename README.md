# FUTURE_DS_03
# 📊 Marketing Funnel & Conversion Performance Analysis

## 📌 Overview
This project analyzes a marketing funnel to understand how users move through the stages:
**Visitors → Leads → Engaged Leads → Customers**.
The Power BI dashboard highlights conversion rates, drop-offs, and key performance trends to support data-driven decisions.

## 🎯 Objectives
- Analyze funnel-stage conversions
- Identify drop-off points
- Evaluate customer behavior across demographics
- Measure overall conversion performance
- Provide actionable insights for optimization

## 🛠 Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Power Query (ETL & Data Cleaning)

## 📂 Dataset Description
The dataset contains marketing and customer information including:
- Funnel stage data (Visitors, Leads, Engaged Leads, Customers)
- Demographics (Age, Education, Job, Marital Status)
- Campaign outcomes and targeting data

## 🧮 Key DAX Measures
- Total Visitors
- Total Leads
- Engaged Leads
- Total Customers
- Visitor-to-Lead Conversion Rate
- Lead-to-Customer Conversion Rate
- Engaged-to-Customer Conversion Rate
- Minimum Conversion
- Maximum Conversion
- Target Conversion

## 📊 Dashboard Pages
- **Funnel Overview:** Visualizes user movement and drop-offs
- **Demographic Analysis:** Conversion trends by age and education
- **Performance KPIs:** Key metrics and conversion gauge

## 📈 Visualizations Used
- Funnel Chart
- Line Chart (Conversion by Age)
- Pie Chart (Customers by Education)
- Gauge Chart (Conversion Performance)
- KPI Cards

## 🔍 Key Insights
- Significant drop-off occurs between Leads and Engaged Leads
- Middle-age groups show higher customer conversion
- Higher education levels contribute to better engagement
- Overall conversion remains below target, indicating optimization scope

## ✅ Conclusion
This dashboard provides a clear and structured view of marketing funnel performance, helping stakeholders identify weaknesses, improve targeting strategies, and increase customer conversion rates.
